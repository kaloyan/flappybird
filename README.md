### FlappyBird Game

Flappy Bird game hack using Javascript HTML and CSS

## How To Play?

You can play the game on [Github pages](https://kaloyan.github.io/flappybird/)

## Available Scripts

In the project directory, you can run:

### `npm install`

Runs the app in the development mode.

### `npm start`

to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.
